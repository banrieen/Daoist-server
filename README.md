## 概述

从本地或远端数据源获取信息，自动进行分析提取。

大模型 AI 它能做，不意味着必然能做
它能做，不意味着正确
它能做，不意味着最优

一个人要有广博，深度的知识，认知能力；能够识别理解问题，提出问题，判别回答才有可能应用好大模型

如大模型是一把神剑，只有拥有拔起神剑的能力才有可能发挥出它的威力；一个普通人不会因碰到了神剑，就成了神。

个人还是要努力学习掌握基础的知识和技能。
